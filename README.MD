# Enhanced Attack Indicator Reborn

Enhanced Attack Indicator Reborn is a **Fabric** and **NeoForge** mod for Minecraft 1.21.11 and later that shows the attack indicator progress on more actions. This is a fork of the original [**Enhanced Attack Indicator**](https://modrinth.com/mod/enhanced-attack-indicator) mod, which is no longer receiving updates for new versions of Minecraft.

Added Actions:
* Breaking blocks
* Bows, crossbow, trident drawing
* Eating foods
* Items with a cooldown (shields, chorus fruit, enderpearls).
* Sleep Progress
* Fullness of Shulker Boxes / Bundles

Other Features:
* Ability to enable/disable the above individually
* Ability to change the priority of the normal weapon cooldown
* Ability to disable pickaxe and shovel weapon cooldowns
* Ability to disable axe weapon cooldowns
